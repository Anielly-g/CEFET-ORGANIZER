# Proposta do Trabalho
Com a proposta desse trabalho, o projeto (CEFET-ORGANIZER) consiste em criar um software focado em organizar as horas complementares da rede CEFET-MG dos cursos Engenharia da Computação, Engenharia Mecatrônica e Design de Moda.

# Passo a Passo da Utilização do Software
O sistema foi desenvolvido na plataforma do Visual Studio Code (VSCode). Nessa plataforma, foram baixadas três extensões: ```HTML```, ```CSS Support```, ```JavaScript (ES6) code snippets``` e ```Live Server```. Essas extensões são essenciais para o funcionamento do software na web.

Para obter a visualização na web após a instalação do VSCode, das extensões e de possuir o código disponibilizado no computador, a pessoa deverá ir no arquivo "index.html" no Visual Studio e clicar no "Go Live" no canto inferior direito. Ele direcionará para uma página Web.

Depois de configurar o VSCode, instalando as devidas extensões e abrindo a pasta de origem para compilar o código, o usuário deverá abrir o arquivo ```index.html``` dentro do próprio VSCode e clicar em ```Go Live```, conforme é mostrado na imagem logo abaixo.

- A função desse dessa página inicial consiste, basicamente, em escolher entre ```Aluno``` e ```Administrador```.
